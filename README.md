# William Ferros Engineering Portfolio

## Publish with GitHub Pages

1. Delete the old contents of your `Engineering-Portfolio` repository, or upload these files over them.
2. Upload **the contents of this folder**, not the outer `william-ferros-portfolio` folder.
3. Confirm that `index.html` is visible at the repository root.
4. Open **Settings → Pages**.
5. Choose **Deploy from a branch**.
6. Select `main` and `/(root)`, then save.
7. Your site will appear at:
   `https://williamferros.github.io/Engineering-Portfolio/`

The source files and reports are intentionally included so portfolio visitors can open or download them.
